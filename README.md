# 単純な電卓
[![Netlify Statu](https://api.netlify.com/api/v1/badges/8eda9c93-ce63-4ce8-bb9b-c0d151818216/deploy-status)](https://app.netlify.com/sites/cirkit/overview)

## 機能
- 四則演算
- 小数点対応
- マイナス対応
- キーボード入力対応

## 開発
Node.js環境下
```
$ npm install --save styled-components
```
